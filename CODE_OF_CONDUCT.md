# Code Of Conduct

This project expects respectful, professional, and technically grounded collaboration.

## Expected Behavior

- Be respectful and constructive.
- Focus on evidence, reproducibility, and clear technical reasoning.
- Keep discussions relevant to the repository and the change under review.
- Accept review feedback in good faith.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Publishing private information without permission
- Abusive, deceptive, or intentionally disruptive conduct
- Knowingly introducing secrets or unsafe content into the repository

## Enforcement

The project owner and delegated maintainers may edit, hide, lock, or remove content and may restrict repository participation when conduct violates this policy.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, review threads, and any project spaces directly managed for this repository.
