# Template Modules

This directory contains reusable templates derived from the repository baseline.

## Current Template Set

- `AGENTS.md`: reusable AGENTS template for other repositories that want a similar Codex operating model

## Intended Use

Use these templates when bootstrapping another repository that should inherit the same baseline approach:

- explicit operating rules,
- Serena-first exploration,
- Context7-first external documentation policy,
- structured planning and review expectations.

Templates are not machine snapshots.
They are human-maintained starting points that should be adapted to the target repository after copying.
